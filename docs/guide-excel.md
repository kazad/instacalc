# Excel Compatability

Instacalc supports the most common Excel functions (`SUM`, `PMT`, etc.). Other tips: 

* Formulas like `=R1 * 3` work. (The leading `=` isn't needed, but muscle memory!) 
* Use row ranges like `SUM(R1:R3)` as you'd expect
